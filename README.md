# MCP is dead, long live MCP!

This project is a proof of concept to build out an MCP server for code execution to scale up the number of tools and reduce token usage. Modeled after the paradigms discussed in the following resources. This is also a [presentation](./docs/DECKSET_PRESENTATION.md) written in markdown that can be rendered as a deck using [Deckset](https://www.deckset.com/) the docs are available [here](./docs/DECKSET_DOCS.md).

- [Cloudflare's blog post](./docs/CLOUDFLARE_CODE_MODE_BLOG_POST.md) on code mode is a great resource to understand the problem and the solution.
- [Daytona AI SDK](./docs/DAYTONA_AI_SDK_DOCS.md) is a platform to run TypeScript code in a sandboxed environment.
